Available commands

- /help -> Get help
- /status -> See your configuration
- /add_staking -> Add coin to Staking Watchlist
- /add_saving -> Add coin to Saving Watchlist
- /add_defi -> Add coin to DEFI Watchlist
- /remove_staking -> Remove coin from Staking
   Watchlist
- /remove_saving -> Remove coin from Saving
   Watchlist
- /remove_DEFI -> Remove coin from DEFI
   Watchlist
- /list_staking -> List of available coins
   for Staking
- /list_saving -> List of available coins
   for Saving
- /list_defi -> List of available coins
   for DEFI
- /top_staking -> Top coins for Staking
- /top_saving -> Top coins for Saving
- /top_defi -> Top coins for DEFI Staking
- /me_staking -> My top coins for Staking
- /me_saving -> My top coins for Saving
- /me_defi -> My top coins for DEFI
- /pro -> PRO Subscription
